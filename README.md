# Board Game - Tic Tac Toe & Gomoku

This project implements the **Tic Tac Toe** and **Gomoku** games in Java, following the **MVC (Model-View-Controller)** architecture. The player can choose to play against another human or an AI.

![image](https://github.com/user-attachments/assets/da19eb90-58c4-4fbb-acef-f69e83a608a0)

## Features

- Choose between **Tic Tac Toe** (3x3) and **Gomoku** (15x15).
- Play against a **real player** or **AI**.
- Handles errors and displays game results (winner, draw).
- **MVC Architecture**: Game logic, user interface, and interactions are separated.
